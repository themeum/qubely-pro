#  Qubely Pro

##  Development
- Navigate to reactjs folder `cd  assets/reactjs`
-  `npm install`
-  `npm start`
-  `gulp watch` (for CSS hotloading)

##  To Build
### Javascript
- Navigate to reactjs folder `cd  assets/reactjs`
- run `npm run build &&  gulp sass`

### To Bundle
- Under root folder, install dependencies `npm install`
- run `gulp`, new file `qubely-pro.zip` should appear
